## 📖 معرفی پروژه
پروژه **BlueReferral** یک سیستم کامل و جامع بر بستر تلگرام است که به شما امکان راه‌اندازی یک **فروشگاه آنلاین همراه با سیستم همکاری در فروش (Referral)** را می‌دهد. این سیستم با دارا بودن یک **مینی اپلیکیشن (Mini App)** بسیار مدرن، تجربه کاربری بی‌نظیری (مشابه اپلیکیشن‌های بومی) را برای کاربران شما فراهم می‌کند.

این پروژه صرفاً یک ربات ساده نیست؛ بلکه یک پلتفرم کامل با پشتیبانی از پرداخت‌های ریالی، رمزارزی و Telegram Stars است که به صورت کاملاً مستقل روی سرور شما میزبانی می‌شود.

---

## ✨ امکانات و ویژگی‌های برجسته

### 📱 مینی اپلیکیشن (Mini App) مدرن
- **طراحی خیره‌کننده:** رابط کاربری مدرن با انیمیشن‌های روان و Skeleton Loaders
- **سبد خرید چندگانه:** امکان افزودن چندین محصول به سبد خرید و تسویه یک‌جا
- **نمایش لحظه‌ای:** بازخوانی بلادرنگ اطلاعات بدون نیاز به رفرش (Live Polling برای ادمین)
- کاملاً ریسپانسیو و بهینه‌شده برای موبایل و دسکتاپ (سازگار با وب‌ویو تلگرام)

### 🛒 سیستم فروشگاه حرفه‌ای
- پشتیبانی از محصولات ساده و دارای پلن/متغیر (Variants)
- سیستم مدیریت موجودی (Inventory) دقیق
- سیستم تخفیف پیشرفته (کدهای تخفیف درصدی و مبلغ ثابت)
- پشتیبانی از انواع شیوه‌های تحویل: دستی، اکانت/ایمیل پسورد، فایل، کد لایسنس و لینک ساب‌اسکریپشن

### 💳 درگاه‌ها و روش‌های پرداخت
- **پرداخت ریالی (کارت به کارت):** ثبت فیش واریزی توسط کاربر و تأیید/رد توسط ادمین
- **پرداخت رمزارزی خودکار:** پشتیبانی از USDT، TRX و TON با سیستم بررسی خودکار هش تراکنش (TXID) از طریق کرون‌جاب
- **ستاره تلگرام (Telegram Stars):** ایجاد فاکتور مستقیم و پرداخت سریع با ستاره‌های تلگرام
- **کیف پول داخلی:** امکان شارژ حساب و خرید مستقیم با موجودی کیف پول کاربر

### 💰 سیستم همکاری در فروش و گیمیفیکیشن
- **درخت دعوت (Referral Tree):** پاداش‌دهی برای دعوت دوستان به صورت درصدی یا مبلغ ثابت
- **کد دعوت اختصاصی (Custom Code):** امکان ساخت کدهای دعوت سفارشی برای اینفلوئنسرها
- **گردونه شانس (Spin Wheel):** گردونه شانس با جوایز قابل تنظیم در پنل مدیریت
- **پاداش ورود روزانه (Daily Check-in):** سیستم محاسبه توالی (Streak) و اهدای شانسِ گردونه برای فعالیت مداوم
- **مأموریت‌ها (Missions):** تعیین تسک‌های مختلف برای کاربران جهت کسب موجودی بیشتر

### 🎫 سیستم تیکتینگ و پشتیبانی
- ارسال و دریافت تیکت پشتیبانی مستقیماً از داخل مینی اپلیکیشن
- رابط کاربری شبیه به چت (Messenger-like) بین کاربر و ادمین
- مدیریت وضعیت تیکت‌ها (باز، بسته)

### 👑 پنل مدیریت (Admin Panel) قدرتمند
- **Customer 360:** مشاهده کامل پرونده یک کاربر (موجودی، سفارشات، برداشت‌ها و زیرمجموعه‌ها) در یک نگاه واحد
- سیستم تهیه بکاپ از دیتابیس با یک کلیک و ارسال مستقیم فایل به ربات تلگرام ادمین
- خروجی اکسل (CSV) از محصولات و سفارش‌ها
- افزودن ادمین‌های فرعی با نقش‌های اختصاصی (دسترسی فقط به بخش مالی، فقط سفارش‌ها و...)

## 🛠 پیش‌نیازها
برای نصب و اجرای این پروژه روی سرور شخصی به موارد زیر نیاز است:
- **سیستم عامل:** ابونتو (Ubuntu 20.04 یا جدیدتر) / دبیان
- **وب سرور:** Nginx یا Apache
- **پایگاه داده:** MySQL 8 یا MariaDB
- **زبان برنامه‌نویسی:** PHP 8.1 به بالا (پیشنهاد: PHP 8.3)
- **اکستنشن‌های PHP:** `pdo`, `pdo_mysql`, `curl`, `mbstring`, `json`
- **دامنه / ساب‌دامین متصل به SSL (HTTPS اجباری برای مینی اپ)**

## 🚀 نصب و راه‌اندازی سریع

این پروژه دارای یک اسکریپت نصب تعاملی (CLI) است که تمام مراحل خسته‌کننده ساخت دیتابیس و اتصال وب‌هوک را خودکار انجام می‌دهد.

1. کلون کردن مخزن روی سرور:
   ```bash
   git clone https://github.com/YourUsername/BlueReferral.git
   cd BlueReferral
   ```
2. اجرای اسکریپت نصب:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

**اسکریپت چه کارهایی انجام می‌دهد؟**
- تنظیمات پایگاه داده را دریافت کرده و جداول را وارد (`schema.sql`) می‌کند.
- توکن ربات شما و آی‌دی عددی (Telegram ID) ادمین اصلی را می‌گیرد.
- فایل تنظیمات (`app/config.php`) را به صورت خودکار تولید می‌کند.
- وب‌هوک تلگرام ربات شما را روی دامنه تنظیم می‌کند.
- برای شما یک میانبر دائمی ترمینال (دستور `blue-ref`) می‌سازد.

## ⚙️ مدیریت کرون جاب (Cron Jobs)
جهت چک کردن مداوم و اتوماتیک تراکنش‌های شبکه کریپتو و همچنین وظایف پاکسازی سفارش‌های منقضی شده، فایل `cron_crypto.php` باید به صورت دوره‌ای توسط سرور اجرا شود. اسکریپت نصب معمولاً این کار را می‌کند اما کد کرون جاب در لینوکس به شکل زیر است:

```cron
* * * * * php /path/to/your/project/public/cron_crypto.php >> /dev/null 2>&1
```

## 🛡 امنیت و معماری
- **جلوگیری از تقلب:** تمامی تعاملات مینی اپلیکیشن با سرور در سمت بک‌اند با استفاده از توابع تایید هویت تلگرام (`Telegram Web App Data Validation`) با الگوریتم HMAC-SHA256 اعتبارسنجی می‌شوند.
- **ایزوله بودن فایل‌ها:** فایل‌های حیاتی سیستم (نظیر دیتابیس و تنظیمات در پوشه `app`) خارج از دید مستقیم وب‌سرور (پوشه `public`) قرار دارند.
